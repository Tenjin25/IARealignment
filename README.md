# IARealignment: Iowa County Election Map

This project visualizes county-level election data in Iowa using Mapbox GL JS.

## Features
- Interactive map centered on Iowa
- Ready for county election data visualization

## Setup
1. Get a free Mapbox access token from [mapbox.com](https://account.mapbox.com/access-tokens/).
2. Replace your Mapbox access token in `main.js`.
3. Open `index.html` in your browser.

## Project Structure
- `index.html`: Loads Mapbox GL JS and displays the map
- `main.js`: Initializes the map and is ready for data visualization
- `style.css`: Basic styling for the map and page
- `README.md`: Project overview and setup instructions
